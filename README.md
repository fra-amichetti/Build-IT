# Build-IT
BuildIT – Piattaforma web per la gestione di cantieri edili. Permette ad amministratori e dipendenti di gestire cantieri, fasi, squadre, documenti tecnici/contabili. I clienti accedono in sola lettura per monitorare l'avanzamento. Autenticazione sicura, controllo sovrapposizioni squadre, statistiche economiche e verifica automatica scadenze.
